개인정보처리방침

최종 업데이트: 2025-01-01

개인 개발자 김성훈(이하 “운영자”)는 이용자의 개인정보를 중요하게 생각하며 관련 법령을 준수합니다. 본 방침은 Trip Receipt 서비스(이하 “서비스”)에 적용됩니다.

1. 수집하는 정보

  • 계정 정보: Google 로그인 또는 Apple 로그인(Sign in with Apple) 제공 과정에서 전달되는 정보(예: 이메일, 사용자 식별자). 제공되는 항목은 로그인 제공자 설정 및 이용자 선택에 따라 달라질 수 있습니다.
  • 사용자 입력 정보: 지출 기록, 메모, 장소명 등 서비스 이용을 위해 사용자가 직접 입력한 콘텐츠
  • 앱 이용 정보: 기능 사용 기록, 오류 로그(최소 범위)
  • 기기 정보: OS 버전, 앱 버전 등(품질 개선 및 문제 해결 목적)

2. 이용 목적

  • 서비스 제공 및 계정 기반 데이터 동기화
  • 서비스 품질 개선 및 오류 대응
  • 고객 문의 응대
  • AI 분석 기능 제공 (아래 “외부 AI API” 항목 참고)

3. 보관 및 처리

  • 서비스 제공을 위해 이용자가 입력한 기록(지출/메모 등)은 서버에 저장될 수 있습니다.

4. 외부 서비스(제3자) 및 위탁

4-1. 외부 AI API

본 서비스는 이용자가 요청한 AI 분석 기능을 제공하기 위해, 이용자가 입력한 텍스트(예: 메모/기록 내용 등)를 외부 AI API로 전송하여 처리할 수 있습니다. 전송되는 정보는 기능 제공에 필요한 최소 범위로 제한됩니다.
  • AI 분석 제공을 위해 제3자 서비스가 데이터 처리를 수행할 수 있습니다.
  • 운영자는 기능 제공에 필요한 범위 내에서만 전송·처리되도록 관리합니다.

4-2. 장소 검색 기능

본 서비스는 지도 표시 없이, 사용자가 장소를 검색해 입력할 수 있는 기능을 제공할 수 있으며, 이 과정에서 Google Places API 등 외부 서비스를 사용할 수 있습니다.

5. 이용자의 선택 및 삭제

  • 이용자는 앱 내 제공되는 기능을 통해 자신의 기록을 삭제할 수 있습니다(제공 여부는 앱 버전에 따라 달라질 수 있음).
  • 개인정보는 목적 달성 후 지체 없이 삭제합니다. 단, 법령상 보관 의무가 있는 경우 해당 기간 동안 보관될 수 있습니다.

6. 문의

개인정보 관련 문의: support@tripreceipt.app

  • サービス提供のため、利用者が入力した記録（支出/メモ等）はサーバーに保存される場合があります。

4-1. 外部 AI API

本サービスは、利用者が要求した AI 分析機能を提供するため、利用者が入力したテキスト（例：メモ/記録内容等）を外部 AI API に送信して処理する場合があります。 送信する情報は機能提供に必要な最小範囲に限定します。

4-2. 場所検索機能

本サービスは地図表示なしで、利用者が場所を検索して入力できる機能を提供する場合があり、その際に Google Places API 等の外部サービスを利用することがあります。

  • 利用者はアプリ内の機能により記録を削除できる場合があります（提供状況はアプリバージョンにより異なります）。
